# 腾讯云OpenClaw连接企业微信详细步骤指南

## 📋 前言
本指南详细说明在腾讯云平台上部署OpenClaw并连接企业微信的完整步骤。腾讯云方案以界面友好、操作简单著称，适合新手用户。

## 🎯 方案特点
- **一键部署**：无需复杂配置
- **可视化面板**：图形化界面操作
- **快速接入**：几分钟完成部署
- **全面支持**：企业微信、QQ、钉钉、飞书四大IM

## 📝 准备工作

### 1. 所需账号
- **腾讯云账号**：需完成实名认证
- **企业微信账号**：需要有管理后台访问权限
- **API Key**：腾讯混元大模型API Key（可选，首次开通送100万tokens）

### 2. 预算预估
- **服务器费用**：约50-100元/月（2核2G配置）
- **模型费用**：混元大模型有免费额度
- **总成本**：最低约50元/月起

## 🚀 第一阶段：购买和部署OpenClaw服务器

### 步骤1：访问腾讯云OpenClaw部署页面
1. 打开浏览器，访问：https://curl.qcloud.com/eWZtc3QF
2. 或访问：https://cloud.tencent.com/act/pro/lighthouse-moltbot
3. 点击"立即购买"或"一键部署"

### 步骤2：选择服务器配置
1. **地域选择**：
   - 国内用户：选择"广州"、"上海"等国内地域
   - 海外用户：选择"香港"、"新加坡"等海外地域

2. **套餐选择**：
   - **入门级**：2核2G 3M带宽（约50元/月）
   - **推荐级**：2核4G 4M带宽（约80元/月）
   - **性能级**：4核8G 5M带宽（约150元/月）

3. **镜像选择**：
   - 在"应用镜像"中选择"OpenClaw（Clawdbot）"
   - 确认镜像版本为最新

### 步骤3：完成购买
1. 选择购买时长（建议1个月起）
2. 确认订单信息
3. 完成支付
4. 等待服务器创建完成（约1-3分钟）

### 步骤4：初始配置
1. 进入腾讯云控制台：https://console.cloud.tencent.com/lighthouse
2. 找到刚购买的服务器实例
3. 点击实例名称进入详情页

## 🚀 第二阶段：配置OpenClaw

### 步骤5：访问OpenClaw管理界面
1. 在服务器详情页，找到"应用管理"或"应用详情"
2. 点击"访问OpenClaw控制台"
3. 系统会自动打开新标签页

### 步骤6：初始设置
1. **首次访问设置**：
   - 设置管理员用户名和密码
   - 保存好登录信息

2. **API Key配置**：
   - 进入"设置" → "模型配置"
   - 选择"腾讯混元大模型"
   - 输入API Key（首次开通有免费额度）
   - 或选择其他支持的模型

3. **基础功能测试**：
   - 在对话界面输入"你好"测试
   - 确认能正常回复

### 步骤7：获取服务器信息
1. 回到腾讯云控制台
2. 在服务器详情页找到：
   - **公网IP地址**：如 `123.123.123.123`
   - **端口号**：OpenClaw服务端口（通常是随机生成）
   - **访问Token**：在OpenClaw控制台获取

## 🚀 第三阶段：企业微信配置

### 步骤8：创建企业微信智能机器人

#### 8.1 登录企业微信管理后台
1. 访问：https://work.weixin.qq.com/wework_admin
2. 使用企业微信管理员账号登录

#### 8.2 进入智能机器人管理
1. 左侧导航栏 → "应用管理"
2. 找到"智能机器人"或"自建应用"
3. 点击"创建应用"

#### 8.3 创建机器人应用
1. **应用信息**：
   - 应用名称：如"OpenClaw助手"
   - 应用logo：可上传自定义图标
   - 应用介绍：简要描述功能

2. **可见范围**：
   - 选择可以使用该机器人的部门或成员
   - 建议先选择测试部门

3. **开发信息**：
   - 应用类型：选择"机器人"
   - 开发模式：选择"API模式"

#### 8.4 获取配置信息
1. 在应用详情页找到：
   - **AgentId**：应用ID
   - **Secret**：应用密钥
   - **企业ID**：CorpID

2. **重要**：复制并保存这些信息

### 步骤9：配置回调URL

#### 9.1 构造回调URL
根据腾讯云服务器的信息构造URL：
```
http://<公网IP>:<端口号>/webhooks/wecom
```
示例：`http://123.123.123.123:8080/webhooks/wecom`

#### 9.2 在企业微信中配置
1. 回到企业微信应用管理
2. 找到"接收消息"配置
3. 点击"设置API接收"

#### 9.3 填写API配置
1. **URL**：填入上一步构造的回调URL
2. **Token**：随机生成或自定义（保存好）
3. **EncodingAESKey**：随机生成（保存好）
4. **消息加密方式**：选择"安全模式"

#### 9.4 保存并验证
1. 点击"保存"
2. 系统会发送验证请求到回调URL
3. OpenClaw会自动处理验证
4. 显示"配置成功"表示验证通过

## 🚀 第四阶段：OpenClaw端配置

### 步骤10：在OpenClaw中配置企业微信

#### 10.1 登录OpenClaw控制台
1. 使用之前设置的管理员账号登录
2. 进入控制台界面

#### 10.2 进入插件管理
1. 左侧菜单 → "插件管理"
2. 找到"企业微信插件"
3. 点击"配置"

#### 10.3 填写企业微信信息
1. **基础配置**：
   - CorpID：企业ID
   - AgentId：应用ID
   - Secret：应用密钥

2. **回调配置**：
   - Token：在企业微信设置的Token
   - EncodingAESKey：在企业微信设置的Key

3. **功能配置**：
   - 消息类型：全选（文本、图片、文件等）
   - 自动回复：开启
   - 群聊支持：开启

#### 10.4 保存并启用
1. 点击"保存配置"
2. 启用插件
3. 重启OpenClaw服务（如有提示）

## 🚀 第五阶段：测试和验证

## 🛠️ 常见问题解决

### Q1：企业微信验证失败
**症状**：配置API接收时显示验证失败

**解决方案**：
1. 检查回调URL格式是否正确
2. 确认服务器端口已开放
3. 检查Token和EncodingAESKey是否一致
4. 确认OpenClaw服务正常运行

### Q2：机器人无响应
**症状**：发送消息后机器人不回复

**排查步骤**：
1. 检查企业微信机器人是否已添加到群聊
2. 确认@了正确的机器人
3. 查看OpenClaw服务日志
4. 检查网络连接

### Q3：消息延迟高
**症状**：机器人回复很慢

**优化建议**：
1. 升级服务器配置
2. 选择离用户近的地域
3. 优化OpenClaw配置
4. 检查网络带宽

### Q4：文件传输失败
**症状**：无法发送或接收文件

**解决方案**：
1. 检查文件大小限制
2. 确认文件类型支持
3. 检查存储空间
4. 查看错误日志
