## 区块链项目综合分析报告 (2021-2025)

### 前言

本报告旨在对 Credmark, Apricot Finance, RMRK, Jambo Network, Endaoment, Mirror 这六个在2021年获得融资的区块链项目进行深入研究。通过分析这些项目在2021年的融资情况以及后续发展，并结合最新的行业动态，力求全面评估它们的发展潜力、创新模式以及对区块链生态系统的贡献。本报告的数据来源主要包括项目方的官方信息、行业媒体报道、社交媒体讨论以及公开的区块链数据。

### 一、Credmark：DeFi风险建模的开创者

#### 1.1 项目概况（2021年）

Credmark 是一家加密信用评级公司，2021 年获得了 550 万美元的投资，旨在与 API3 预言机合作推出一个去中心化的风险模型平台，用于对 DeFi 协议的风险进行评级。

#### 1.2 最新进展（2021-2025）

*   **产品开发与迭代：**

    *   推出 SmartPool 工具，帮助用户在 Uniswap V3 上构建仓位，根据用户的风险偏好选择价格范围。
    *   与 Alchemist Coin 合作推出 Crucible Rewards 计划，通过 CMK-ETH SLP 代币进行订阅。
*   **社区建设与激励：**

    *   举办黑客马拉松，鼓励开发者为 Uniswap v3 提供流动性策略。
    *   推出社区更新，分享 SmartPool 更新、数据平台进展、代币经济学等信息。
    *   进行内容创作比赛，鼓励社区参与。
*   **合作伙伴关系：**

    *   与 API3 预言机合作推出去中心化的风险模型平台。
    *   与 Bridge Mutual 合作，为 DeFi 提供保险服务。
*   **去中心化治理：**

    *   朝着 DAO（去中心化自治组织）的方向发展，实现社区和运营的自动化。
*   **安全审计：**

    *   强调数据完整性，解决区块链数据高可用、低规范化的问题。

#### 1.3 媒体报道

*   Blockonomi: 报道 Credmark 通过风险建模平台改变 DeFi 游戏。
*   新闻报道强调了 Credmark 在 DeFi 数据和风险评估方面的重要性。

#### 1.4 项目分析

Credmark 旨在通过构建去中心化的风险模型平台，为 DeFi 用户提供更可靠的风险评估工具，这对于 DeFi 生态系统的健康发展至关重要。

#### 1.5 风险与挑战

*   **模型准确性：** 风险模型的准确性是关键，需要不断优化和验证。
*   **数据质量：** 依赖链上数据，数据质量直接影响模型结果。
*   **市场接受度：** DeFi 用户是否接受并使用该平台进行风险评估，存在不确定性。
*   **DeFi 复杂性：** DeFi 协议的复杂性给风险建模带来挑战。

### 二、Apricot Finance：Solana生态的借贷创新者

#### 2.1 项目概况（2021年）

Apricot Finance 是 Solana 生态上的一个借贷平台，旨在提供个性化的 DeFi 解决方案。2021 年完成了种子轮和 A 轮融资，总额超过 480 万美元。

#### 2.2 最新进展（2021-2025）

*   **测试网发布与迭代：**

    *   发布测试网，用户可以体验 X-Farm 和 Apricot Assist 等功能。
    *   持续改进测试网，解决用户反馈的问题。
*   **产品创新：**

    *   推出 X-Farm，允许用户存入任何资产并以高达 3 倍的杠杆开始耕种。
    *   Apricot Assist 自动去杠杆，降低清算风险。
*   **合作伙伴关系：**

    *   与 Pyth Network、Switchboardxyz 等合作，提高数据可靠性和安全性。
    *   与 Liq Protocol 合作，实现链上清算。
    *   与 Gate.io、MEXC Global 等交易所合作。
*   **社区建设：**

    *   招聘营销主管，扩大团队。
    *   举办 AMA 活动，与社区互动。
    *   推出 Apricot NFT，激励社区参与。

#### 2.3 媒体报道

*   媒体报道强调了 Apricot Finance 在 Solana 生态系统中的地位和创新。

#### 2.4 项目分析

Apricot Finance 通过 X-Farm 和 Apricot Assist 等创新功能，旨在提高用户的收益和降低风险，符合 Solana 生态的发展趋势。

#### 2.5 风险与挑战

*   **智能合约风险：** 借贷平台面临智能合约漏洞的风险。
*   **市场波动：** 加密货币市场波动可能导致清算风险。
*   **Solana 网络稳定性：** Solana 网络拥堵和不稳定可能影响平台运行。
*   **竞争：** Solana 生态借贷平台竞争激烈。

### 三、RMRK：波卡生态NFT的革新者

#### 3.1 项目概况（2021年）

RMRK 是一个波卡生态的 NFT 基础设施，旨在建立跨链 NFT 基础设施标准。2021 年获得了 600 万美元融资。

#### 3.2 最新进展（2021-2025）

*   **协议开发与应用：**

    *   发布 RMRK 2.0，允许 NFT 在任何时间点进行表情和扩展功能。
    *   Kanaria 平台提供多资源 NFT 交易。
    *   开发 Sub.id，用于查看 RMRK NFT。
*   **NFT 市场：**

    *   推出 Singular.rmrk.app NFT 市场，提供经济高效的 NFT 铸造和交易体验。
*   **社区建设：**

    *   举办艺术比赛，奖励社区成员。
    *   通过社交媒体与社区互动，分享项目进展和行业洞察。
*   **生态合作：**

    *   与 Bifrost、Kilt Protocol 等项目合作，扩展 NFT 的应用场景。

#### 3.3 媒体报道

*   新闻报道强调了 RMRK 在波卡生态 NFT 领域的影响力和创新。

#### 3.4 项目分析

RMRK 通过其创新的 NFT 基础设施，为波卡生态带来了更强大的 NFT 功能，如多资源、嵌套和可组合性，具有广阔的应用前景。

#### 3.5 风险与挑战

*   **技术复杂性：** RMRK 的技术架构相对复杂，需要开发者和用户具备一定的专业知识。
*   **波卡生态发展：** 波卡生态的发展速度直接影响 RMRK 的发展。
*   **NFT 市场竞争：** NFT 市场竞争激烈。
*   **跨链互操作性风险：** 跨链互操作性可能面临安全风险。

### 四、Jambo Network：区块链广告技术的探索者

#### 4.1 项目概况（2021年）

Jambo Network 是一个基于波卡生态的区块链广告技术支持平台，旨在实现透明、可信的去中心化广告产业价值生态。2021 年完成了 200 万美元机构轮融资。

#### 4.2 最新进展（2021-2025）

*   **技术开发：**

    *   构建开放的技术平台，基于 Substrate 框架，为项目开发者维护了许多可定制的功能和模块。
*   **价值提升：**

    *   持续深化链上流量价值，提升流量精准度。
*   **平台愿景：**

    *   使广告行业的服务网络能够为行业提供透明、高效、稳定的通信服务。

#### 4.3 媒体报道

*   媒体报道相对较少。
#### 4.4 项目分析

Jambo Network 旨在解决广告行业中存在的不透明、欺诈等问题，具有一定的市场潜力。

#### 4.5 风险与挑战

*   **市场接受度：** 广告行业是否接受区块链技术存在不确定性。
*   **数据隐私：** 如何在保护用户隐私的前提下提供精准广告，是一个挑战。
*   **竞争：** 区块链广告平台竞争激烈。
*   **波卡生态发展：** 波卡生态的发展速度直接影响 Jambo Network 的发展。

### 五、Endaoment：DeFi慈善事业的先锋

#### 5.1 项目概况（2021年）

Endaoment 是一个链上非营利慈善组织，建立在以太坊上，旨在通过 DeFi 实现慈善捐赠。2021 年完成了 250 万美元种子轮融资。

#### 5.2 最新进展（2021-2025）

*   **基金管理：**

    *   支持开立捐助者建议基金，使用加密货币支持慈善事业。
*   **合作与赠款：**

    *   与 Rari Capital 合作，由 RGT 持有人投票决定捐赠对象。
    *   向 Stand With Asians 社区基金捐赠超过 62.5 万美元。
*   **碳抵消：**

    *   推出可减税的直接行动碳抵消基金，接受多种加密货币。
*   **社区建设：**

    *   通过社交媒体与社区互动，分享项目进展和慈善事业。
*   **安全审计：**

    *   链上智能合约经过 OpenZeppelin 的审计。

#### 5.3 媒体报道

*   Decrypt: 报道 Endaoment 完成 250 万美元种子轮融资。

#### 5.4 项目分析

Endaoment 通过其链上慈善组织，为加密货币持有者提供了一种便捷的捐赠方式，并促进了慈善事业的发展。

#### 5.5 风险与挑战

*   **加密货币波动：** 加密货币价格波动可能影响基金价值。
*   **监管风险：** 慈善捐赠可能面临监管政策的不确定性。
*   **公众认知：** 如何提高公众对链上慈善的认知度，是一个挑战。
*   **税务问题：** 加密货币捐赠的税务问题需要解决。

### 六、Mirror：去中心化内容发布的平台

#### 6.1 项目概况（2021年）

Mirror 是一个去中心化内容发布平台，旨在为创作者提供代币发行工具，帮助创作者众筹。2021 年估值达到 1 亿美元。

#### 6.2 最新进展（2021-2025）

*   **写作体验升级：**

    *   推出全新的编辑器，改善写作体验。
*   **社区激励：**

    *   空投 WRITE 代币给 Mirror 最活跃的成员。
    *   举办 WRITE Race，鼓励作家参与。
*   **产品创新：**

    *   引入 Editions，允许创作者发布 NFT 作品。
    *   推出拆分功能，将资金分配给多个以太坊地址。
    *   引入拍卖功能。
*   **众筹：**

    *   支持以太坊纪录片众筹，筹集超过 200 万美元。
*   **平台价值：**

    *   Mirror 上的数据存储在 Arweave 上，实现去中心化存储。

#### 6.3 媒体报道

*   The Information：报道 Union Square Ventures 投资 Mirror。

#### 6.4 项目分析

Mirror 通过其去中心化的内容发布平台和代币发行工具，为创作者提供了一种新的创作和众筹方式，具有一定的创新性。

#### 6.5 风险与挑战

*   **内容质量：** 如何保证平台上内容的质量，是一个挑战。
*   **用户获取：** 如何吸引更多的创作者和读者使用该平台，存在不确定性。
*   **市场竞争：** 内容发布平台竞争激烈。
*   **去中心化存储：** Arweave 的存储成本和可靠性可能影响用户体验。

### 综合分析

这六个项目分别代表了区块链领域不同的应用场景，包括 DeFi 风险评估、Solana 生态借贷、波卡生态 NFT 基础设施、区块链广告、链上慈善和去中心化内容发布。它们在各自的领域都进行了一定的创新，并为区块链生态系统的发展做出了贡献。

### 总结

Credmark, Apricot Finance, RMRK, Jambo Network, Endaoment, Mirror 这六个项目在 2021 年都获得了投资，并在后续发展中取得了一些进展。其中，Credmark 致力于提供 DeFi 风险评估；Apricot Finance 专注于 Solana 生态的借贷创新；RMRK 在波卡生态构建 NFT 基础设施；Jambo Network 尝试构建区块链广告平台；Endaoment 旨在推动链上慈善；Mirror 则致力于去中心化内容发布。
