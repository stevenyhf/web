

### 专题研究：2022年NFT100社区领袖系列综合分析报告

**研究对象：Matty, Manifold, Loopify, LooksRare, Lisa Mayer, Lesley Silverman**

#### 一、文档主要内容总结

提供的文档为2022年撰写的《NFT100社区领袖系列》研究内容，涵盖了六位NFT领域的重要社区领袖或实体：Matty、Manifold、Loopify、LooksRare、Lisa Mayer 和 Lesley Silverman。这些文档基于Twitter数据和公开信息，详细记录了每位领袖或实体的背景、活动领域、社区贡献及其对NFT生态的影响。以下是对这些领袖相关文档内容的总结：

##### 1. Matty (@DCLBlogger)
- **背景与角色**：Matty 是 NFT 领域的内容创作者和社区领袖，2018年5月加入Twitter，拥有22.5万关注者，创立了 The Meta Key 项目，专注于 NFT 教育和元宇宙发展。
- **主要观点与活动**：分享 NFT 行业颠覆多个领域的潜力（如艺术、虚拟土地、游戏），通过 YouTube 和 Twitter 提供教育内容，开发 The Meta Key 作为元宇宙体验平台，讨论市场周期和投资策略（文档N2459）。
- **社区影响**：通过教育内容和项目开发，Matty 降低了新用户进入 NFT 的门槛，推动了元宇宙概念的普及。

##### 2. Manifold (@manifoldxyz)
- **背景与角色**：Manifold 是一个 Web3 创意主权平台，2021年3月加入Twitter，拥有5.9万关注者，致力于为创作者提供 NFT 铸造和智能合约工具。
- **主要观点与活动**：推出 Manifold Studio 为创作者提供免费铸造工具，支持批量铸造、动态 NFT、链上版税，与 Coinbase NFT 等平台合作，优化创作者体验（文档N2458）。
- **社区影响**：Manifold 通过技术工具降低了创作者进入 Web3 的成本，支持了艺术家和开发者的创意主权。

##### 3. Loopify (@Loopifyyy)
- **背景与角色**：Loopify 是 NFT 项目 Treeverse 和 Interleave 的创始人，2017年5月加入Twitter，拥有19.3万关注者，专注于 NFT 游戏和社区建设。
- **主要观点与活动**：分享 NFT 致富策略和市场见解，开发 Treeverse 作为 MMORPG 游戏，强调社区文化和长期价值，讨论 NFT 版税和市场趋势（文档N2457）。
- **社区影响**：通过游戏项目和市场分析，Loopify 推动了 NFT 游戏领域的创新，增强了社区对实用性的关注。

##### 4. LooksRare (@LooksRare)
- **背景与角色**：LooksRare 是一个 NFT 交易市场，2021年8月加入Twitter，拥有23.7万关注者，旨在通过交易奖励和低费用挑战 OpenSea。
- **主要观点与活动**：提供交易奖励和版税即时支付，支持 EIP-2981 链上版税，优化 UI 和 API 功能，举办社区活动和标志设计比赛（文档N2456）。
- **社区影响**：LooksRare 通过奖励机制和低成本交易吸引了用户，增强了 NFT 市场的竞争和创作者收入。

##### 5. Lisa Mayer (@lisacmayer)
- **背景与角色**：Lisa Mayer 是 Boss Beauties NFT 的 CEO 和 My Social Canvas 的创始人，2009年2月加入Twitter，拥有1.8万关注者，专注于女性赋权和 NFT 社区建设。
- **主要观点与活动**：通过 Boss Beauties NFT 激励女性和女孩，与 Barbie 和 Marvel 合作推广女性角色，组织读书俱乐部和导师活动，分享个人创业和母亲经历（文档N2455）。
- **社区影响**：Lisa Mayer 通过 NFT 项目和活动增强了女性在 Web3 中的参与，推广了包容性和社会价值。

##### 6. Lesley Silverman (@lesleysilverman)
- **背景与角色**：Lesley Silverman 是 UTA（United Talent Agency）的 Web3 负责人，2009年2月加入Twitter，拥有1,887名关注者，专注于 NFT 在娱乐和艺术领域的应用。
- **主要观点与活动**：推动 UTA 与 CryptoPunks 和 Deadfellaz 等 NFT 项目合作，组织拍卖和活动（如 Sotheby’s 合作），讨论 NFT 对创作者经济的意义，参与行业会议（如 CES 和 SXSW）（文档N2454）。
- **社区影响**：Lesley Silverman 通过将主流娱乐与 NFT 结合，扩大了 NFT 在传统行业的曝光和应用。

- **合规与全球化**：数字人民币等法定数字货币的普及，可能推动NFT交易合规化，特别是在中国市场。

###### 5.2 挑战

- **政策引导**：制定NFT和元宇宙资产法律框架，支持数字人民币支付，平衡创新与监管。

- **展望**：六位领袖的多样化路径表明，NFT生态将在实用性、文化价值和元宇宙融合方向持续演进，未来需协作构建安全、包容、可持续的Web3生态。
