### 🤖 Assistant

向量数据库（Vector Database）是专门用于存储、检索高维向量（如文本、图像、音频的嵌入表示）的数据库，支持高效的相似性搜索。以下是主流向量数据库的详细介绍及使用步骤。

## **三、使用步骤详解（以Pinecone为例）**
### **1. 安装客户端库**
```bash
pip install pinecone-client
```

### **2. 初始化Pinecone**
```python
import pinecone

# 初始化（需API密钥，从官网获取）
pinecone.init(api_key="YOUR_API_KEY", environment="us-west1-gcp")

# 创建索引（维度需与嵌入向量一致）
index_name = "rag-demo"
dimension = 768  # 例如BERT嵌入的维度
pinecone.create_index(index_name, dimension=dimension, metric="cosine")

# 连接到索引
index = pinecone.Index(index_name)
```

### **3. 插入向量数据**
```python
# 示例：插入ID为"vec1"的向量
vectors = [
    ("vec1", [0.1, 0.2, ..., 0.768]),  # 假设是768维向量
    ("vec2", [0.3, 0.5, ..., 0.768])
]
index.upsert(vectors=vectors)
```

### **4. 相似性搜索**
```python
# 查询与目标向量最相似的Top-K结果
query_vector = [0.15, 0.25, ..., 0.768]  # 示例查询向量
results = index.query(vector=query_vector, top_k=3, include_values=True)

# 输出结果
print(results)
# 返回示例：{"matches": [{"id": "vec1", "score": 0.95}, ...]}
```

### **5. 删除索引（可选）**
```python
pinecone.delete_index(index_name)
```

## **四、其他数据库快速示例**
### **1. Milvus（本地部署）**
```python
from pymilvus import connections, Collection

# 连接服务
connections.connect("default", host="localhost", port="19530")

# 创建集合（类似表）
collection = Collection("demo_collection")
collection.search(
    data=[[0.1, 0.2, ...]],  # 查询向量
    anns_field="embedding",   # 向量字段名
    param={"metric_type": "L2", "params": {"nprobe": 10}},
    limit=5
)
```

### **2. Chroma（轻量级本地使用）**
```python
import chromadb

# 创建客户端
client = chromadb.Client()

# 创建集合
collection = client.create_collection("demo")

# 插入数据
collection.add(
    documents=["This is a document", "Another document"],
    embeddings=[[0.1, ...], [0.2, ...]],  # 可自动生成（需模型）
    ids=["id1", "id2"]
)

# 查询
results = collection.query(query_texts=["Find similar docs"], n_results=2)
```

## **五、关键注意事项**
1. **向量维度一致性**：确保插入的向量维度与索引定义一致。
2. **距离度量选择**：
   - `cosine`：文本相似性（默认）。
   - `L2/Euclidean`：图像/音频相似性。
3. **生产环境优化**：
   - 使用GPU加速（如Milvus的CUDA支持）。
   - 分布式部署（如Milvus集群）。
4. **成本控制**：
   - Pinecone按向量数量计费，小型项目可用免费版。
   - 开源方案需自建基础设施。

## **六、典型应用场景**
1. **RAG（检索增强生成）**：从向量库检索文档片段输入LLM。
2. **推荐系统**：通过用户/商品嵌入向量匹配相似项。
3. **语义搜索**：替代关键词搜索（如“找与‘人工智能’相关的论文”）。
4. **去重/聚类**：快速识别相似内容（如新闻去重）。

---

通过选择合适的向量数据库，结合业务需求（数据规模、实时性、成本），可以高效实现语义搜索和AI增强应用。
