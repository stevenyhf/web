# 企业微信连接OpenClaw详细配置部署步骤

## 📋 前言
本指南详细说明如何通过企业微信连接使用OpenClaw，尽量不涉及代码和CLI部署，重点放在界面操作和配置步骤上。

## 🎯 准备工作

### 1. 所需资源清单
- **企业微信账号**：需要有企业微信管理后台访问权限
- **云服务器**：阿里云、腾讯云或百度云的轻量应用服务器（已预装OpenClaw）
- **API Key**：百炼（阿里云）或其他AI模型的API Key
- **域名**（可选）：用于解决企业微信域名校验问题

### 2. 选择云服务商（三选一）
- **阿里云**：功能最完整，文档最详细
- **腾讯云**：界面友好，操作简单
- **百度云**：预装插件，配置较少

## 🚀 方案一：阿里云方案（最详细）

### 第一阶段：购买和部署OpenClaw服务器

#### 步骤1：购买轻量应用服务器
1. 访问阿里云控制台：https://swasnext.console.aliyun.com/buy
2. 选择"应用镜像" → 选择"OpenClaw"镜像
3. 配置服务器：
   - **实例**：内存必须2GiB及以上
   - **地域**：建议选择"华北2（北京）"
   - **购买时长**：根据需求选择
4. 完成购买

#### 步骤2：配置OpenClaw
1. 进入服务器管理页面
2. 点击"应用详情"页签
3. 执行以下操作：

**a. 端口放通**
- 点击"端口放通"下的"执行命令"
- 系统会自动开放OpenClaw服务端口
- **重要**：端口是随机生成的，需要记下端口号

**b. 配置API Key**
- 点击"配置OpenClaw"下的"执行命令"
- 选择API Key配置方式：
  - **系统推荐**：选择离服务器最近的API Key
  - **手动输入**：输入自己的百炼API Key
- 选择API Key对应地域

**c. 获取访问信息**
- 点击"访问控制页面"下的"执行命令"
- 获取OpenClaw WebUI地址
- （可选）查看并保存Token

### 第二阶段：企业微信配置

#### 步骤3：创建企业微信智能机器人

**a. 访问企业微信管理后台**
1. 登录：https://work.weixin.qq.com/wework_admin
2. 左侧导航栏 → "应用管理" → "智能机器人"

**b. 创建机器人**
1. 点击"创建机器人"
2. 选择"手动创建"
3. 选择"API模式创建"

**c. 获取Token和EncodingAESKey**
1. 分别点击"Token"和"EncodingAESKey"后的"随机获取"
2. **重要**：复制并保存这两个值

#### 步骤4：配置OpenClaw插件（推荐方式）

**a. 在阿里云控制台配置**
1. 回到阿里云服务器管理页面
2. 在"企业微信配置"区域填入：
   - Token：上一步获取的Token
   - EncodingAESKey：上一步获取的EncodingAESKey
3. 点击"应用"

**b. 获取回调URL**
1. 在阿里云控制台查看OpenClaw的：
   - 服务器公网IP地址
   - 端口号（随机生成）
2. 构造回调URL格式：
   ```
   http://<服务器IP>:<端口号>/webhooks/wecom
   ```
   例如：`http://123.123.123.123:8080/webhooks/wecom`

**c. 完成机器人创建**
1. 回到企业微信管理后台
2. 在API模式创建页面填入：
   - URL：上一步构造的回调URL
   - Token：之前保存的Token
   - EncodingAESKey：之前保存的EncodingAESKey
3. 点击"创建"

### 第三阶段：验证和测试

#### 步骤5：测试连接
1. 在企业微信中创建一个群聊
2. 点击"添加群成员"
3. 搜索刚创建的机器人名称，加入群聊
4. 在群聊中@机器人发送消息测试

#### 步骤6：解决域名校验问题（如遇到）

如果出现"域名主体校验未通过"错误：

**a. 配置自有域名**
1. 访问阿里云AppFlow域名管理：https://appflow.console.aliyun.com/domain/manage
2. 点击"添加域名"
3. 输入二级域名，如：`airobot.yourcompany.com`

**b. 配置DNS解析**
1. 在域名管理平台添加CNAME记录
2. 将二级域名解析到AppFlow提供的记录值

**c. 更新回调URL**
1. 使用新域名替换原来的IP地址
2. 新URL格式：`http://airobot.yourcompany.com/webhooks/wecom`

## 🔧 高级配置

### 1. 更改AI模型
1. 进入OpenClaw WebUI
2. 点击"设置"图标
3. 搜索"Primary Model"
4. 更改为其他模型，格式：`alibaba-cloud/模型Code`
5. 例如：`alibaba-cloud/qwen3-max-2026-01-23`

### 2. 配置定时任务

**a. 在企业微信中配置**
1. 进入机器人设置
2. 打开"消息推送"
3. 复制Webhook地址

**b. 在AppFlow中配置定时任务**
1. 访问AppFlow控制台
2. 使用定时任务模板创建连接流
3. 配置：
   - 执行频率（cron表达式）
   - 要执行的指令
   - OpenClaw服务器信息
   - 企业微信Webhook地址
4. 发布连接流

### 3. 安全配置
1. **关闭公网访问**：配置完成后可关闭WebUI公网访问
2. **Token保护**：不要泄露包含Token的完整URL
3. **防火墙**：仅开放必要的端口

## 🛠️ 常见问题解决

### Q1：域名校验未通过怎么办？
**解决方案**：
1. 确保使用企业自有域名
2. 域名需已完成备案
3. 备案主体需与企业微信企业主体一致或有关联

### Q2：机器人无响应怎么办？
**排查步骤**：
1. 检查服务器网络是否正常
2. 确认端口已正确放通
3. 验证Token和EncodingAESKey是否正确
4. 检查回调URL格式是否正确

### Q3：消息发送失败怎么办？
**检查项**：
1. 企业微信机器人是否已添加到群聊
2. 是否使用了正确的@方式
3. 服务器资源是否充足
4. API Key是否有剩余额度

### Q4：如何查看OpenClaw日志？
1. 进入服务器管理页面
2. 查看应用日志
3. 或通过WebUI查看运行状态
