🛠️ 安装步骤

pip install https://github.com/Panniantong/agent-reach/archive/main.zip

2️⃣ 完整自动安装（推荐）
# 完整自动化安装：会自动安装系统依赖（gh CLI、Node.js、mcporter、xreach 等）
agent-reach install --env=auto
说明：--env=auto 会检测当前系统并尝试安装缺失的依赖。如果您不希望自动改动系统，请使用 安全模式（第 3 步）。
3️⃣ 安全模式（不自动安装系统包）
如果您在生产环境或不想让脚本自行安装系统软件，使用以下命令：
agent-reach install --env=auto --safe
安全模式只会检查已安装的依赖并报告缺失项，不会自行执行 apt-get、npm install -g 等系统级操作。
您可以根据 agent-reach doctor 给出的缺失列表手动安装所需软件。
4️⃣ 预览（Dry‑Run）— 查看将要执行的操作
agent-reach install --env=auto --dry-run
该命令仅列出将要安装或配置的内容，不进行任何更改。阅读输出后再决定是否继续。
5️⃣ 检查安装状态
agent-reach doctor
doctor 会列出所有 渠道（channel） 的状态，显示 ✅（正常）或 ❌/⚠️（有问题）。
若有红灯，请根据提示逐项处理；如需进一步帮助，请把输出粘给我。
